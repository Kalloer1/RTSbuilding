RTS Building: Build From Above
Update Summary for 1.0.2-beta / 0.0.2-beta

1.0.2-beta is the Minecraft 1.21.1 NeoForge build.
0.0.2-beta is the Minecraft 1.20.1 Forge build.

Main changes:
- Added a manual storage refresh button.
  - The RTS storage panel now has an R button next to the help button.
  - The compact inventory overlay also has an R button next to the help button.
  - Use it when you want to force linked storage to rescan immediately.

- Reduced inventory-open lag in large modpacks.
  - The compact overlay no longer forces a full linked-storage scan every time
    a container or creative inventory screen opens.
  - Existing local storage data is reused until the user performs an explicit
    storage action such as refresh, search, sort, or paging.
  - Creative tab classification is warmed and cached to avoid repeated heavy
    rebuilds in large modpacks.

- Added storage scan feedback.
  - Storage refresh requests now show a small progress indicator while waiting
    for the server response.

- Improved ultimine responsiveness.
  - Chain mining no longer performs all follow-up block breaks and drop imports
    in one large server tick after the first block finishes.
  - Chained blocks are processed in small batches, reducing visible freezes.

- Fixed overlay localization issues.
  - Added the missing return/import label translation in the Forge 1.20.1 build.
  - Updated overlay help and bottom-bar guide text for English, Simplified
    Chinese, Traditional Chinese Taiwan, and Traditional Chinese Hong Kong.
